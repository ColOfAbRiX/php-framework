build_login  [line 33]

String build_login( String $text, Integer $length  )

Crea la form per il login e ne gestisce gli eventi

La funzione crea la form per il login degli amministratori e ne effettua il login se la form è gia stata inviata. Se il login ha avuto successo vengono valorizzate le variabili di sessione "is_logged" e "user_id" contenenti la prima una variabile booleana che indica il successo del login e la seconda l'identificativo dell'utente sul database.

 Parameters:
String   $text:  Il testo da tagliare.
Integer   $length:  La lunghezza a cui tagliare il testo.

API Tags:
Return:  Il testo tagliato.

check_user  [line 88]

Int check_user( String $user, String $pwd  )

Effettua il login di un amministratore

Tramite lo username e la password come argomenti la funzione è ingrado di determinare se quell'utente è presente sul database. Se l'utente è presente viene restituito il suo id. Se l'utente non fosse presente la funzione attende due secondi prima di terminare per prevenire attacchi brute-force e restituisce il valore zero.

 Parameters:
String   $user:  Il nome utente
String   $pwd:  La password

API Tags:
Return:  Zero se l'utente non è presente, il suo ID del database altrimenti
